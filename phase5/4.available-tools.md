# Solaris Web Agent - Available Tools

## **Agent Browser Tools** (4 tools)
- **agentBrowserNavigate** - Navigate to URL and return interactive snapshot with element refs (@e1, @e2)
- **agentBrowserInteract** - Click, fill, type, hover, check/uncheck elements using refs from snapshot
- **agentBrowserExtract** - Extract text/html/value/snapshot from page elements or full page
- **agentBrowserClose** - Close browser session and cleanup resources

## **Web Tools** (2 tools)
- **browseWeb** - Browse web pages and extract text content (Browserbase with fallback)
- **webSearch** - Search internet using Brave Search API with DuckDuckGo fallback

## **Document Management Tools** (8 tools)
- **createDocument** - Create new document artifact (text, code, sheet, presentation, webview)
- **listDocuments** - List user documents with optional kind filtering and project scoping
- **readDocument** - Read full content of a document by ID
- **searchDocuments** - Search documents by title/content with kind filtering and project scoping
- **updateDocument** - Update existing document with AI-powered changes
- **requestSuggestions** - Get writing suggestions for existing documents
- **createPlan** - Create structured plan artifact for multi-step tasks
- **readPlan** - Read current state of plan document
- **updatePlan** - Update plan progress, mark steps complete, add notes

## **Code Execution Tools** (1 tool)
- **executeCode** - Execute Python code in E2B sandbox with file generation, pip packages, charts

## **Utility Tools** (1 tool)
- **getWeather** - Get current weather by city name or coordinates (requires approval)

## **Total: 16 Tools**

All tools are available to thinking models and can be selectively enabled/disabled by category through the tool settings UI. Document tools are scoped to the active project when one is selected.
